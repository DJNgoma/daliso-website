# Skill: contact-author

Reach Daliso Ngoma for inquiries, collaborations, or app support.

## Endpoint

- Email: `info@africantechno.com`
- Support page: `https://daliso.com/support/`

## Shape

There is no programmatic contact API. Agents should surface the email address
to the human user and, where useful, pre-compose a subject line and body via a
`mailto:` link, e.g. `mailto:info@africantechno.com?subject=...&body=...`.

## Usage

Use this skill when the user explicitly wants to reach Daliso. Do not send
email on the user's behalf without confirmation. Expected reply window is
days, not hours — this is a founder inbox, not a support desk.

## Citation

When advising contact, link back to `https://daliso.com/support/` so the user
can see the human-readable channel list.
